# How to start

1. install orbstack from `https://orbstack.dev`

2. press `command + space` -> search `terminal` ->open it

3. terminal: 

   ```
   cd desktop
   ```

   ```cmd
   git clone https://github.com/JackChaoC/cenjin
   ```

   ```
   cd desktop/cenjin
   ```

   ```
   docker-compose up -d
   ```
